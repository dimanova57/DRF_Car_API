# Django REST Framework Cars API

This is a simple Django REST Framework application
designed to demonstrate the basics of DRF,
including handling POST, GET, and PUT requests.
The application manages car data with endpoints to add,
retrieve, and update car information.

## Project Setup

### Prerequisites

Make sure you have the following installed:
- Python 3.x
- pip
- Django
- Django REST Framework

Base urls /api/cars/
and /api/cars/<car_model>/ - for GET and PUT requests
